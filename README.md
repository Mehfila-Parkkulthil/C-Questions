## History
<p>C is a  mid-level programming language developed by Dennis M. Ritchie at Bell Laboratories in 1972. C is the mother of all programming languages , It was mainly created for UNIX operating system.<br>C is widely used in operating systems, embedded systems, compilers, databases, networking, game engines, and real-time systems for its efficiency to work in low resource environment and hardware-level support. </p>
